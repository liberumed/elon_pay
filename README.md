## Installation
1) Copy ``config/example-config.exs`` to ``config/secret.exs`` and provide local configuration
2) Run: ``mix deps.get``
3) Run: ``mix run priv/repo/seeds.exs``
3) Run: ``mix run priv/repo/seed_case_1.exs``
4) Run: ``mix run --no-halt``
